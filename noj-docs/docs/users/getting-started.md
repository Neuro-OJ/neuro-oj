# 快速开始

## 登录

打开 NOJ 前端站点后，可以注册新账号或使用运营者提供的账号登录。若账号被要求首次改密，系统会先引导你完成密码修改。

## 查找题目

进入题目列表后，可以按题号、标题、难度、题型或分类筛选题目。题目详情页包含题面、输入输出说明、限制条件和提交接口。

## 阅读提交接口

NOJ 题目通常会在题面中声明需要实现的函数。例如 A+B 题要求：

```python
def solve(a: int, b: int) -> int:
    ...
```

这表示评测端会调用 `solve(a, b)`，而不是运行传统的 stdin/stdout 主程序。

## 进入提交页

在题目详情页选择语言并填写代码。提交后可以在提交详情页查看排队位置、评测状态、分数、输出和结构化详情。
